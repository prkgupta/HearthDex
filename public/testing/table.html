Testing Table

An <table> is used to define a table.

An <th> is used to define a header cell in a table.

An <tr> is used to define a row in a table.

An <td> is used to define a cell in a table.

An <caption> is used to define a table caption.

An <colgroup> is used to specify a group of one or more columns in a table for formatting.

An <col> is used to specify column properties for each column within a <colgroup> element.

An <thead> is used to group the header content in a table.

An <tbody> is used to group the body content in a table.

An <tfoot> is used to group the footer content in a table.
