Testing HTML

Since HTML has many other elements, it will be useful to explore the rest of the element we may need some day.

To do this we have created some simple example that demonstrate the following HTML5 element in a valid way:

  • <blockquote>
  • <embed>
  • <iframe>
  • <ol>
  • <q>
  • <strong>
  • Table: <table>, <th>, <tr>, <td>, <caption>, <colgroup>, <col>, <thead>, <tbody>, <tfoot>
  • <video>
